# Hotel Reservation

這是一個使用 **FastAPI** 建立的訂房網站後端專案。

---

##  🚀前端啟動方式

### 1️⃣ npm install (需安裝node.js)
```bash
npm install
```
### 2️⃣ 啟動server
``` bash
npm run dev
```

## 🚀 後端啟動方式

### 1️⃣ 建立虛擬環境
```bash
python -m venv .venv
.venv\Scripts\activate
```
### 2️⃣ 安裝套件

```bash
pip install -r requirements.txt
```
